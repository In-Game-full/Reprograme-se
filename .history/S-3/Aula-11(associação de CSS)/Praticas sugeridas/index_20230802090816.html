Biblioteca Bootstrap

A biblioteca bootstrap é um kit de ferramentas front end que fornece componentes para desenvolvimento web responsivos e mobile first com HTML, CSS e JavaScript prontos
- Layouts: Contém modelos pré-definidos de design como grids, carouse ols, navbars etc.
- Componentes: Fornecerá classes
- Estilos: Possui uma coleção robusta de estilos pre definidos para elementos HTML básicos (como textos, links) até form
